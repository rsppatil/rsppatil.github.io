Misc. resources:

Some useful bash resources:

  • explainshell.com is a useful website where you can input a bash command with flags (for instance the rsync command rsync -havzP /src/ /dest/), and it pulls a description for the command and its flags from the manpages.
  • The Unix Shell course shell-novice by Software Carpentry offers a nice beginner friendly hands-on introduction to bash shell commands and shell scripts.
